# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-02-21

### Added
- Multi-LLM provider system (OpenAI, Gemini, Ollama)
- 8 built-in content templates (blog, social, email, SEO, ads, etc.)
- API key management with rate limiting
- SSE streaming for real-time content generation
- Web dashboard with dark theme
- Content export (Markdown, plain text, JSON)
- SQLite database for usage tracking
- Docker + Docker Compose support (with Ollama)
- GitHub Actions CI/CD (lint, test, Docker build)
- Comprehensive test suite
